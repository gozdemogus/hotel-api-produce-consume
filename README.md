# Hotel Reservation API - Core MVC Project 
Aimed to create an API consumer project through the hotel reservation - promotion site concept.

## Used

- C# & .NET Core Web API & .NET Core MVC
- Entity Framework
- JavaScript, Ajax, JQuery
